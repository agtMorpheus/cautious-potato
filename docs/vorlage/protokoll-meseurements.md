# Protokoll Messungen (Measurements)

## Überstrom-Schutzeinrichtung & Fehlerstrom-Schutzeinrichtung

### Spalten (Columns)

| Spalte | Bezeichnung | Beschreibung |
|--------|-------------|--------------|
| A | Pos.Nr. | Positionsnummer |
| B | Nr. | Nummer |
| C | Zielbezeichnung | Zielbezeichnung |
| I | Kabel Typ | Kabeltyp |
| K | Leiter Anzahl | Anzahl der Leiter |
| M | Leiter Querschnitt | Querschnitt der Leiter |
| O | Un | Nennspannung |
| P | fn | Nennfrequenz |
| Q | Art Charakteristik | Art der Charakteristik |
| S | In (A) | Nennstrom in Ampere |
| U | ZS(Ω) L-PE | Schleifenimpedanz L-PE in Ohm |
| V | ZN(Ω) L-N | Impedanz L-N in Ohm |
| W | Ik (kA) L-PE | Kurzschlussstrom L-PE in kA |
| Y | Riso (MΩ) ohne Verbraucher | Isolationswiderstand ohne Verbraucher in MΩ |
| Z | Riso (MΩ) mit Verbraucher | Isolationswiderstand mit Verbraucher in MΩ |
| AB | RPE (max. 1Ω) | Schutzleiterwiderstand (max. 1Ω) |
| AC | GEWISS RCD | GEWISS RCD |
| AD | In (A) | Nennstrom in Ampere |
| AF | I∆n (mA) | Nenn-Fehlerstrom in mA |
| AG | Imess (mA) | Gemessener Strom in mA |
| AI | Auslösezeit tA (ms) | Auslösezeit in ms |
| AK | UL≤50V Umess (V) | Gemessene Spannung (≤50V) in V |
| AN | Diff. Strom (mA) | Differenzstrom in mA |
